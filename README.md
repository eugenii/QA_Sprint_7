# QA_Sprint_7
## Самокат

### Проект тестирования сервиса самоката.

адрес сервиса: https://qa-scooter.praktikum-services.ru/

API документация: https://qa-scooter.praktikum-services.ru/docs/


Реализованные методы:

- Создание курьера

- Удаление курьера

- Получение id курьера по логину и имени

- Логин курьера

-----


- создание заказа

- получение списка заказов

Реализованные тесты:

работа с курьерами:

- успешное и неуспешное создание курьера (неполные, невалидные данные, повторяющиеся данные)
  
- логин курьера
  
- удаление курьера (дополнительное)
  

работа с заказами:

- создание заказа
  
- получение списка заказов


__Генерация отчета allure: pytest --alluredir=./allure_results__
__Запуск отчета: allure serve ./allure_results__
